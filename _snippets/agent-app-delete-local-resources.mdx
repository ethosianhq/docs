## Delete local resources

Play around and stop the workspace using:

<CodeGroup>

```bash terminal
ethosian ws down
```

```bash full options
ethosian ws down --env dev --infra docker
```

```bash shorthand
ethosian ws down dev:docker
```

</CodeGroup>

or stop individual Apps using:

<CodeGroup>

```bash app
ethosian ws down --group app
```

```bash api
ethosian ws down --group api
```

```bash database
ethosian ws down --group db
```

</CodeGroup>
