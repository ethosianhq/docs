## Delete AWS resources

Play around and then delete AWS resources using:

<CodeGroup>

```bash terminal
ethosian ws down --env prd --infra aws
```

```bash shorthand
ethosian ws down prd:aws
```

</CodeGroup>

or delete individual resource groups using:

<CodeGroup>

```bash app
ethosian ws down --env prd --infra aws --group app
```

```bash api
ethosian ws down --env prd --infra aws --group api
```

```bash database
ethosian ws down --env prd --infra aws --group db
```

</CodeGroup>
