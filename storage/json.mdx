ethosian supports using local JSON files as a storage backend for Agents using the `JsonFileAgentStorage` class.

## Usage


```python storage.py
from ethosian.agent import Agent
from ethosian.tools.duckduckgo import DuckDuckGo
from ethosian.storage.agent.json import JsonFileAgentStorage

agent = Agent(
    storage=JsonFileAgentStorage(path="tmp/agent_sessions_json"),
    tools=[DuckDuckGo()],
    add_history_to_messages=True,
)

agent.print_response("How many people live in Canada?")
agent.print_response("What is their national anthem called?")
```
