title: Introduction

Here you'll find examples that'll help you use ethosian, from basic **agents and workflows** to advanced **fine-tuning and evaluations**. If you have more, please [contribute](https://github.com/ethosianhq/docs) to this list.

You can run each recipe individually or clone the [ethosian cookbook](https://github.com/ethosianhq/ethosian/tree/main/cookbook) and run it from there.
